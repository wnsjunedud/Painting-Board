# Painting-Board
Simple painting board with JS

## What I used
- Javascript
- HTML5
- Canvas
- 2D Context
- Pointer Events
- CSS

## Features
- Paint
- Change Brush Size
- Fill Canvas
- Change Color
- Save Images
- Mouse Events
